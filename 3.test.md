# 测试与分数

## 如何进行测试

我们采用了gtest进行测试，CMake会自动从Github克隆仓库，因此请保证至少你的Terminal能够访问Github

如果无法进行正常测试，请和助教联系

在build目录中执行`.\ftp_test`即可进行测试

要求:
1. Server可执行程序和Client可执行程序名称需要严格按照`ftp_server`和`ftp_client`命名
2. Server和Client的可执行程序需在build目录根目录中
